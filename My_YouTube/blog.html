var:- The scope of var is functional scope. so a variable declared in a function with var is only available for this function. It can be updated and re declar into the scope. The scope is global when var is declared outside a function. used var keyword , if must support for older browsers.
let:- The scope of let is block scope. so a variable declared in a block with let is only available for this block. It can be updated but can not re declar into the scope.
const:- The scope of const is block scope. so a variable declared in a block with const is only available for this block. It can not be updated and re declar into the scope. Used const keyword if the value should not be changed.
